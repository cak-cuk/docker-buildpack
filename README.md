This is base image, contain common package for builing and version control like:
cvs, bazaar, git, subversion

## Avaliable distro:
- Centos7
- Ubuntu LTS (14.04, 18.04, 20.04 and 22.04)
- Debian (Debian 9, 10, 11, and 12)
- Alpine
